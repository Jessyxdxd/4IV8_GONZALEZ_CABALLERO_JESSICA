Objetivo del Portafolio

El presente portafolio de evidencias tiene como objetivo consolidar, organizar y demostrar las competencias técnicas adquiridas en el diseño y desarrollo de aplicaciones web integradas. A través de este espacio, se exhibe el dominio en la creación de interfaces de usuario interactivas, la lógica de programación en el entorno del cliente y la persistencia de datos mediante la conexión estructurada con sistemas de bases de datos.

Elementos Aprendidos en la Materia

HTML5 & CSS

Maquetación semántica, diseño estructural limpio y estilización avanzada utilizando metodologías modernas y un control de paletas de color y diseño responsivo.

JavaScript

Desarrollo de lógica del lado del cliente, manipulación del DOM, control de eventos y resolución de desafíos algorítmicos para hacer la interfaz interactiva.

Conexión a Bases de Datos

Comprensión del flujo de datos entre el cliente y el servidor, gestión de entornos locales/remotos y la correcta implementación de operaciones CRUD bajo arquitecturas ordenadas.
